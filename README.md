Project Descriptoin:

Employee Performance Analysis Dashboard helps organizations track and improve performance by integrating data from HR systems, reviews, and surveys. It visualizes key metrics like goal achievement, competencies, and engagement, with filters, drill-downs, and alerts, enabling data-driven decisions while ensuring privacy and secure access.
